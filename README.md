# 📊 Telco Customer Churn Prediction

## Overview
This project is an end to end Machine Learning application that predicts customer churn in the telecom industry.

It includes:
- Data preprocessing & feature engineering
- Model training using Random Forest
- Deployment using Streamlit

---

## Business Problem
Customer churn directly impacts revenue. This project helps identify customers at risk so companies can take proactive retention actions.

---

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

---

## Features
- Interactive web application
- Real-time predictions
- Encoded feature pipeline
- Clean UI

---
